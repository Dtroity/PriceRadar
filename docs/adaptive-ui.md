# Адаптивный UI (общепит / планшеты / мобильные)

## Брейкпоинты (`tailwind.config.js`)

| Ключ | Ширина | Назначение |
|------|--------|------------|
| `sm` | 640px | Крупный телефон |
| `md` | 768px | Планшет портрет |
| `lg` | 1024px | Планшет альбом / малый ноутбук |
| `xl` | 1280px | Desktop |
| `2xl` | 1536px | Широкий desktop |

## Хук `useBreakpoint()`

`frontend/src/hooks/useBreakpoint.ts` — возвращает `'mobile' | 'tablet' | 'desktop'`:

- **mobile**: `< 768px`
- **tablet**: `768px – 1023px`
- **desktop**: `≥ 1024px`

## Навигация

- **`≥ lg`**: левый сайдбар **240px**, полные подписи (`Sidebar.tsx`).
- **`md – lg-1`**: колонка **64px**, иконки + `title` для подсказки; гамбургер в шапке открывает **overlay** 240px.
- **`< md`**: сайдбар скрыт, **`BottomNav`** (5 пунктов) + **`MoreMenuDrawer`** («Ещё»).

Отступ контента под нижнюю панель: `pb-[calc(4.5rem+env(safe-area-inset-bottom))]` в `Layout.tsx` на `main`.

## Tap targets

Классы в `index.css`: `.tap-target`, `.tap-target-row`. Компактные кнопки в desktop-таблицах: `.table-action-icon` (сброс min-size при `lg+`).

## Зависимости

- `@use-gesture/react` — свайп в списке документов (`SwipeableRow`).
- `react-pull-to-refresh` — обновление списка документов на мобильном.

## Ключевые экраны

| Экран | Поведение |
|-------|-----------|
| Документы | Таблица / карточки + pull-to-refresh + swipe |
| Документ | Desktop: сетка превью + таблица; планшет: скролл таблицы, коллапс превью; мобильный: карточки строк + bottom sheet |
| Закупки | Карточки заявок на мобильном |
| Dashboard | Статистика 2×2, изменения цен — карточки на мобильном |
| Аномалии | Карточки с крупным % на мобильном |
| Аналитика | Чипы периода со скроллом; выбор товара — bottom sheet на мобильном; таблица поставщиков — 3 колонки на мобильном |
